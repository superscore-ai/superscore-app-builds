# Superscore App Builds

Public release artifacts for [**Superscore**](https://superscore.ai) — a macOS menu bar app that turns your voice into instant text at your cursor, in any application.

This repo only hosts downloadable binaries (.dmg and .zip). The source code is private; you don't need it to install or use the app.

## Download

Get the [latest release](https://github.com/superscore-ai/superscore-app-builds/releases/latest), or use the stable direct URLs:

- **macOS DMG** → https://github.com/superscore-ai/superscore-app-builds/releases/latest/download/Superscore.dmg
- **macOS ZIP** → https://github.com/superscore-ai/superscore-app-builds/releases/latest/download/Superscore.zip

Both are signed with a Developer ID and Apple-notarized — no "unidentified developer" warnings.

## Requirements

- macOS 13.3 or later
- Apple Silicon (M1, M2, M3, M4) — Intel is not currently supported

## Install

1. Download the DMG
2. Open it, drag **Superscore** to your Applications folder
3. Launch and grant Microphone + Accessibility when prompted
4. Press your hotkey (default `F5`) to start, press again to stop
